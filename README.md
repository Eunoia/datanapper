# Datanapper

Your gem for taking a nap.

## Installation

Add this line to your application's Gemfile:

    gem 'datanapper'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install datanapper

## Usage

The best time to take a nap is today. To start a quick nap, `DataNapper.nap(1)`
